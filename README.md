# 🌤️ Weather App

A simple weather application built using HTML, CSS, and JavaScript. It uses OpenWeatherMap API to show real-time weather data.

## Features
- Search weather by city
- Temperature and humidity
- Weather conditions
- Wind speed and feels like temperature

## Tech Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

- ## Project structure
├── index.html      # Main webpage structure
├── style.css       # UI design and styling
├── script.js       # Weather API logic (JavaScript)
└── README.md       # Project description for GitHub


## How to Use
1. Enter city name
2. Click search
3. Get weather details instantly
